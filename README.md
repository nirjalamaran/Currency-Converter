# Currency-Converter
A simple and efficient Currency Converter application that allows users to convert values between different currencies in real-time using the latest exchange rates.

Features
Convert between multiple international currencies
Real-time exchange rate updates (via API)
User-friendly and responsive interface
Fast and accurate calculations
Supports popular currencies like USD, INR, EUR, GBP, etc.

 Tech Stack
Frontend: HTML, CSS, JavaScript
API: Exchange Rate API

 How It Works
Enter the amount you want to convert
Select the source and target currencies
Instantly get the converted value
